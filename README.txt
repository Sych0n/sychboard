SychBoard - Electron App
========================

FIRST TIME SETUP
----------------
1. Open Command Prompt in this folder
2. Run: npm install
3. Wait for it to finish (downloads Electron)

TO RUN THE APP (without building)
-----------------------------------
npm start

TO BUILD THE .EXE INSTALLER
-----------------------------
npm run build

The installer will appear in the "dist" folder.

NOTE: You need an icon.ico file in the assets folder.
You can convert the icon.svg to .ico at: https://convertio.co/svg-ico/
Or just run without an icon for now - it will still work.

ADDING YOUR GROQ API KEY
--------------------------
Open the app → Settings (gear icon) → paste your Groq API key
